- 모든 portrait 파일명은 '{일련번호}.png' 형식으로 통일되어 있음
- 각 포켓몬마다 normal(일반), dizzy(기절) 두 종류의 portrait가 있음

============================
!! IMPORTANT NOTICE !!
============================

- normal 버전은 모든 일련번호에 대응되는 파일이 존재하나, dizzy 버전은 일부 누락됨 (자료 존재하지 않음)
- 이 경우, **dizzy 이미지가 없는 포켓몬은 normal 버전을 사용**할 것
- 기절시 css 스타일링을 이용해 UI에 흑백으로 띄울 것이므로, normal 버전을 사용하더라도 상태 구별 가능
